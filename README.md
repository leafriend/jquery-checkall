jQuery-checkAll
===============

jQuery-checkAll is [jQuery](http://jquery.com) plugin to check or not all checkbox group.

    <script>
    $("#checkAll").checkAll();
    </script>
    
    <label for="checkAll">Check All</lable>
    <input type="checkbox" id="checkAll">
